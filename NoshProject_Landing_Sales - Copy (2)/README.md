# NoshProject — Landing Page (Sales)

## Chạy nhanh
- Cách đơn giản nhất: mở `index.html` bằng Chrome/Edge.
- Khuyến nghị (để tránh lỗi cache): dùng VS Code → cài **Live Server** → Right click `index.html` → **Open with Live Server**.

## Cấu trúc thư mục
- `index.html` — nội dung trang (bám sát flow bán hàng)
- `styles.css` — giao diện (cam “giòn” + xanh biển Busan), responsive
- `script.js` — tương tác (menu mobile, countdown, chọn vị, social proof, form validate)
- `assets/` — hình pack 4 vị + mascot + minh hoạ

## Ghi chú
- Tuyến đường Busan → Việt Nam là minh hoạ (không dùng bản đồ địa giới) để tránh tranh cãi.
- Các số “45 calo”, “0% dầu” đang theo **định vị**. Khi có số liệu chính thức, bạn chỉ cần sửa text tại phần Hero & Benefits.
